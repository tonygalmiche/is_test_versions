# is_test_versions

Module Odoo 19 développé par [InfoSaône](http://www.infosaone.com).

## Description

Ce module ajoute un champ **Prénom** (`firstname`) sur la fiche contact (`res.partner`).

Le champ est visible uniquement pour les contacts de type personne (non société) et apparaît juste après le champ **Nom** dans le formulaire.

## Dépendances

- `base`

## Installation

1. Copier le module dans le répertoire des addons.
2. Mettre à jour la liste des modules dans Odoo.
3. Installer le module **is_test_versions**.
